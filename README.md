💼 **Freelance Client Management System:**

A modern React-based web application designed to help freelancers manage clients, track payments, and monitor project earnings efficiently.
This project simulates a real-world freelance workflow where users can add clients, record payments, calculate pending amounts automatically, and store data locally — ensuring information remains saved even after refreshing the browser.

✨ **Project Overview:**

Freelancers often struggle to keep track of multiple clients, payments, and pending balances.
This application solves that problem by providing a simple and intuitive dashboard to manage freelance work in one place.
The system focuses on clean UI, practical business logic, and real-world usability using React fundamentals.

🚀 **Key Features:**

✅ Client Management

- Add new clients with project value

- Edit client details anytime

- Delete clients when projects are completed

💰 Payment Tracking

- Add payments for each client

- Prevent overpayment automatically

- Real-time earnings updates

📊 Smart Dashboard

- Total number of clients

- Total earnings calculated automatically

- Pending amount overview

💾 Persistent Data Storage

- Uses localStorage to save data

- Information remains available after page refresh

🎯 User Experience

- Disabled payment button when project is fully paid

- Clean and responsive layout

- Interactive buttons & input focus effects

⚙️ **Installation & Setup:**

- Clone the repository:

      git clone https://github.com/saumya-30-verma/Freelance-client-management-system.git

- Navigate to project directory:

      cd Freelance-client-management-system

- Install dependencies:

      npm install

- Start development server:

      npm run dev

- Open browser at:

      http://localhost:5173
